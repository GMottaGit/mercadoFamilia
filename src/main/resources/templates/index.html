Bem-vindo ao Sistema de Gestão do Mercado da Família

Este sistema tem como objetivo facilitar o gerenciamento de produtos e funcionários do mercado.

Funcionalidades disponíveis:

  • Cadastrar produtos: Adicione novos itens ao estoque do mercado.
  • Atualizar estoque: Mantenha o controle das quantidades disponíveis.
  • Consultar produtos: Veja a lista de todos os produtos cadastrados.
  • Cadastrar funcionários: Registre os colaboradores que trabalham no mercado.
  • Consultar funcionários: Acesse uma lista com os dados dos funcionários.

Utilize o menu acima para navegar entre as funcionalidades do sistema.
